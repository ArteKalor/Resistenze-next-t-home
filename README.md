Segui questi 3 passaggi:
⭢Cerca il pulsante verde con scritto <> Code nella parte in alto a destra di questa pagina.
⭢Cliccaci sopra e, dal piccolo menu che si apre, clicca su Download ZIP.
⭢Il tuo computer scaricherà un archivio compresso. Una volta terminato il download, fai clic destro sul file scaricato e scegli "Estrai tutto..." (o un'opzione simile) per aprire la cartella e vedere tutti i file pronti all'uso.
